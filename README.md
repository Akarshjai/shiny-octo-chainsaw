# shiny-octo-chainsaw

## About this repository

This repository was created from GitHub's official **["Introduction to GitHub"](https://github.com/skills/introduction-to-github)** Skills course template. It is a learning exercise, not an application — there is no source code, build, or runtime here.

The course walks through the core GitHub workflow:

- Creating a branch
- Committing a file
- Opening a pull request
- Merging a pull request

The `.github/steps/` directory contains the course's step-by-step instructions, and `.github/workflows/` contains the automation the course itself uses to track progress through those steps (it is not project CI/CD). `PROFILE.md` is leftover scaffolding from the course's optional "profile README" exercise. `images/` holds the screenshots used in the course instructions.

## Tech stack

None. This repository contains no application code, package manifest, or build tooling.

## Status

Archived learning exercise — kept for reference, not under active development. If you're looking to take the course yourself, start at [github.com/skills/introduction-to-github](https://github.com/skills/introduction-to-github).
